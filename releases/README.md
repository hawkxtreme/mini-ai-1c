### Версия 1.1.22 — 23 марта 2026

- В шаблоны slash-команд и сценарии с MCP теперь можно быстрее вставлять инструменты прямо из интерфейса.
- В настройках MCP появился отдельный просмотр доступных инструментов по каждому серверу.
- Исправлены дубликаты MCP-инструментов и обновление состояния отключённых серверов.
- Снижено поведение runtime MCP, которое могло выглядеть подозрительно для антивирусов.
