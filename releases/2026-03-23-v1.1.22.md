# Новости релиза 1.1.22

Дата: 23 марта 2026

## Кратко

- В интерфейсе появились быстрые вставки MCP-инструментов в slash-команды и связанные сценарии.
- Для MCP-серверов добавлен отдельный просмотр доступных инструментов и их состояния.
- Backend стал стабильнее обрабатывать список MCP-инструментов: меньше дубликатов, лучше обновление состояния и меньше ложных AV-срабатываний.

## Что нового

### Новые возможности

- В редактор шаблонов slash-команд добавлена вставка MCP-инструментов прямо в текст шаблона.
- В настройках MCP появился отдельный просмотр доступных `Tools` для каждого сервера.
- В связанных UI-сценариях добавлен более удобный выбор инструментов MCP без ручного набора имён.

### Исправления

- Исправлено применение draft MCP settings в tool pickers.
- Убраны дубликаты MCP-инструментов при сборе списка инструментов.
- Улучшено обновление состояния отключённых MCP-серверов.

### Улучшения

- Добавлено агрегирование и кэширование списка MCP-инструментов на backend.
- Снижено runtime-поведение MCP, которое могло выглядеть подозрительно для антивирусных проверок.
- Интерфейс MCP стал прозрачнее: видно, какие инструменты доступны у конкретного сервера и в каком состоянии они находятся.
